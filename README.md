# UID Lab - User Interface Design Experiments

A collection of interactive user interface design experiments and demonstrations.

## Overview

This repository contains various UI/UX design experiments showcasing different aspects of web interface design, including:

- Interactive forms with real-time validation
- Color palette generators
- CSS animations and transitions
- Responsive design patterns
- User interaction feedback

## Structure

```
240701198-UID_Lab/
│
├── index.html              # Main landing page
├── css/
│   └── styles.css          # Main stylesheet with design system
├── js/
│   └── main.js             # Utility functions and common JavaScript
├── experiments/
│   ├── experiment1.html    # Interactive Form with Validation
│   ├── experiment2.html    # Color Palette Generator
```

## Experiments

### Experiment 1: Interactive Form
- Real-time form validation
- User feedback with error and success messages
- Field-level validation for email, phone, and required fields
- Responsive form design

### Experiment 2: Color Palette Generator
- Generate random color palettes
- Multiple color scheme options (Monochromatic, Complementary, Analogous, Triadic)
- Click-to-copy color codes
- Interactive color exploration

### Experiment 3: CSS Animations
- Showcase of various CSS animation effects
- Interactive animation triggers
- Hover effects demonstrations
- Animation control panel
